# CyberDestin RO

A Cyberpunk-themed Tarot reading application built with Next.js, Three.js, and GSAP.

## Features Implemented

- [x] **Project Initialization**
    - [x] Next.js 15 App Router setup
    - [x] TypeScript configuration
    - [x] TailwindCSS v4 integration
    - [x] Custom font setup (Orbitron, Rajdhani, Inter)

- [x] **Internationalization (i18n)**
    - [x] `next-intl` integration
    - [x] Language switching support (English, Romanian, Chinese)
    - [x] Middleware for locale routing

- [x] **3D Card Deck**
    - [x] Three.js / React Three Fiber scene setup
    - [x] Custom 3D Card Mesh with Holographic materials
    - [x] Neon edge glowing effects
    - [x] Post-processing (Bloom)

- [x] **Animations**
    - [x] GSAP integration for high-performance animations
    - [x] "Explosion" effect on reveal
    - [x] "Chaos Shuffle" animation
    - [x] "Reassembly" animation with bounce effect

- [x] **Game Logic**
    - [x] Daily reading limit (1 per day)
    - [x] State management with Zustand
    - [x] Tarot data structure with multilingual support

- [x] **UI/UX**
    - [x] Cyberpunk aesthetic (Neon Purple/Cyan theme)
    - [x] Responsive design
    - [x] HTML overlay for game controls

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
